# iCloud Install Link – Signal.shortcut

이 문서는 **IntentCP Signal 단축어** 설치 링크(아이클라우드 공유 링크)와 버전 정보를 관리합니다.

> 단축어 파일은 저장소에도 포함되어 있습니다:
> - `shortcuts/Signal.shortcut` : 실제 사용/배포용 단축어
> - `shortcuts/Signal (Example).shortcut` : 설정 구조 참고용 예시 단축어
>
> GitHub에서 직접 설치는 불가능하므로, 설치는 아래 iCloud 링크를 통해 진행합니다.

---

## 최신 버전 (Recommended)

- **Signal v1**
  - iCloud 링크: https://www.icloud.com/shortcuts/c75b978c658d4661a695ca70aaff7078
  - 대상: 일반 사용자 / 실제 사용용 단축어
  - 변경사항:
    - IntentCP v1 URL Scheme 지원
    - Base URL / LLM 프롬프트 사용자 설정 방식

---

## 설치 방법

1. 위 iCloud 링크를 iPhone/iPad에서 엽니다.
2. 단축어 앱에서 **"단축어 추가"**를 눌러 설치합니다.
3. 설치 후 단축어 이름이 `Signal` 인지 확인합니다.
4. (권장) 단축어 이름을 Siri 호출용 이름으로 변경합니다.
   - 기본 이름: `Signal`
   - 한국어 사용자 권장 예시: `시그널`
   - Siri 호출 예: "시리야, 시그널"
5. 첫 실행 시 권한 요청(마이크/받아쓰기/네트워크)을 모두 허용합니다.

자세한 설정 체크리스트는 아래 문서를 참고하세요:
- `install/setup-checklist.md`
